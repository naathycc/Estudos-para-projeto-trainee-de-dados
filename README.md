## Projeto Trainee de Dados - CATI Jr.

Este repositório documenta meus estudos durante o Projeto Trainee de Dados da CATI Jr., focando em análise de dados, estatística e aprendizado de máquina.

## Ferramentas Utilizadas

- Python
- Visual Studio Code
- Google Colab
- GitHub
- SQL
- Pandas
- NumPy
- Matplotlib
- Streamlit
- Kaggle

## Organização

Durante o trainee, cada semana será dedicada ao estudo de diferentes tópicos da área de Dados, seguindo o cronograma abaixo:

- **Semana 1:** Fundamentos de Python, Estruturas de Dados e Funções
- **Semana 2:** SQL, NumPy e Pandas
- **Semana 3:** Matemática para Dados
- **Semana 4:** Coleta e Tratamento de Dados
- **Semana 5:** Machine Learning Supervisionado
- **Semana 6:** Machine Learning Não Supervisionado
- **Semana 7:** Streamlit
